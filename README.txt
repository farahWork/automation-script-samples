This is a Maven Project. There are multiple way to run the script. Below is one of them
Prerequisites and Setup:
1. Install Java Development Kit (JDK): Ensure you have a compatible JDK installed on your system.
2. Install VS Code
3. Install Apache Maven: Download and install Maven, and configure the MAVEN_HOME environment variable.
4. Install VS Code Extensions:
    - Java Extension Pack: This provides core Java language support, including IntelliSense, debugging, and project management.
    - Maven for Java: This extension adds specific Maven integration, allowing you to manage dependencies, execute goals, and interact with your Maven projects directly within VS Code.
5. In command Line of VS Code 
    - "cd %demoFolderPath"
    - mvn clean
    - mvn compile
    - mvn package
    - mvn install
    - click run on "test_running.java" on method with "@Test" tag, there will be an icon in front of the method.
